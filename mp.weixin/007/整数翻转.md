# 整数翻转
##  首先看下题目
![整数翻转题目](https://gitee.com/mobilecoder/mdnice/raw/master/2020-9-2/1599058145366-image.png)

看到题目之后第一感觉是，这个easy的题目真的太easy了吧，直接用一个`while(x != 0)`配合`求余`和`整除`就能直接搞定了

### 直接上代码
```java
public static int reverse(int x) {
    int ans = 0;
    while(x != 0){
        int mod = x % 10;
        ans = ans * 10 + mod;
        x = x / 10;
    }
    return ans;
}
```
